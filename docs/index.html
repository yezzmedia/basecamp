Compatibility & Requirements

Basecamp is designed to run anywhere modern Laravel development happens. The script automatically verifies your environment before starting.

🐧

Linux & macOS

Native support for all major distributions and Apple Silicon.

🪟

Windows

Supported via WSL2 (Recommended) or Git Bash.

Technical Prerequisites

  • PHP Runtime 8.2+
  • Composer 2.x
  • Node.js / NPM 18+
  • Git Required

CLI Reference

Automate your workflow with powerful command-line arguments.

Argument Description Example
-y, --yes Enable non-interactive mode (skips all prompts) basecamp.sh --yes
--name Set the project name and directory --name=my-app
--template Choose a preset (basecamp, minimal, ai, full) --template=ai
--packages Comma-separated list of extra packages --packages=pulse,horizon
--force Force overwrite existing project directory --force
