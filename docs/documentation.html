💡 What is Basecamp?

Basecamp is a professional bootstrapping tool for Laravel developers. It replaces the repetitive manual steps of setting up a new project with a streamlined, interactive, or fully automated experience.

Built as a hybrid script, it uses a Bash wrapper for universal compatibility and a Node.js core for a rich, interactive CLI experience.

⚙️ The Installation Workflow

1

System Verification

The script checks for PHP, Composer, Node.js, NPM, and Git. It even checks for tool updates to ensure your environment is ready before any file is touched.

2

Template Selection

Choose from pre-defined stacks (Basecamp, Minimal, AI, Full Ops) or go "Custom" for full manual control. Templates are curated by yezzmedia for maximum efficiency.

3

Automated Execution

Basecamp runs laravel new, modifies configurations, installs Composer packages, and runs initial migrations in a single pass.

4

Health Checks & Finalizing

Sets directory permissions, links storage, and creates the first Git commit. A detailed summary with version numbers is presented at the end.

⌨️ Advanced Automation

Basecamp shines in automated environments like CI/CD pipelines or local setup scripts.

Remote Execution

curl -fsSL https://yezzmedia.github.io/basecamp/install | bash

Non-Interactive / CI Mode

curl -fsSL https://yezzmedia.github.io/basecamp/install | bash --yes --template=full

Local Debugging

./basecamp.sh --verbose

🚑 Troubleshooting

"laravel new" command not found ↓

Basecamp will automatically fallback to composer create-project. You don't need to do anything, but some advanced flags might be skipped.

Permission Denied on basecamp.sh ↓

Run chmod +x basecamp.sh to make the script executable before running it.

Windows execution errors ↓

Ensure you are using WSL2 (Recommended) or Git Bash. PowerShell and CMD are not natively supported for .sh scripts.
